# Aprendendo Herança

## Herança é a abstração para aproveitar atributos e métodos de uma outra classe

> - Essa classe que esta provendo atributos e metodos, podemos chamá-la de superclasse
> - Para realizar essa herança em uma superclasse, temos que usar a palavra reservado **extends**
> - Quando uma subclasse herda atributos e métodos da superclasse, não é mais necessário manter meus atributos e métodos escritos na minha subclasse
